# tv_Global 🏆

Plataforma premium para ver partidos de la Copa del Mundo. 

## Características
- Diseño moderno con modo oscuro y glassmorphism.
- Reproductor de video integrado para pruebas de URL.
- Lista interactiva de partidos con animaciones suaves.

## Uso
Abre el archivo `index.html` en cualquier navegador web moderno.
